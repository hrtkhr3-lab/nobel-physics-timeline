# CLAUDE.md — 歴代ノーベル物理学賞 ビジュアル年表

このリポジトリで作業する Claude Code 向けのガイド。ユーザーの意図・設計方針・生成のしくみを記す。

## プロジェクト概要

ノーベル物理学賞の**受賞研究そのもの**を解説する**完全オフラインの自己完結型HTMLツール**。
1901〜2024年の**45件**を古い順に収録。各ページに図解を1つずつ備える。

**2段構え**（2026-07 改修）:
- **やさしい本文**（背景→発見→意義、数式なし、高校生向け）は従来どおり全ページ共通。
- 全45ページ末尾に折りたたみ **「発展」セクション**（`<details class="adv">`、大学初年級、数式あり）。
- 図解は2種類: **体験型シミュレーション（25件）**=物理式に基づく実単位・定量計算
  （sims_data.py）と、**模式図（20件・理論中心の回）**=従来の軽い操作（pages_data.py のまま）。

UI・本文はすべて日本語。やさしい本文では数式を使わない（発展セクションと図解の注記では使ってよい）。

## ファイル構成

```
歴代ノーベル賞/
├── index.html            … 年代順タイムライン一覧（全45件を埋め込み・検索/分野フィルタ）
├── README.md             … 使い方・収録リスト・出典方針（利用者向け）
├── CLAUDE.md             … このファイル
├── laureates/            … 受賞ごとの個別ページ（自己完結HTML・45ファイル）
│   └── NN_YYYY_name.html … 連番=年代順の通し番号
└── build/                … 生成スクリプト（閲覧には不要・メンテ用）
    ├── build_pages.py    … 共通レイアウト雛形(TEMPLATE)＋生成エンジン＋NAV(年代順)＋apply_overrides()
    ├── pages_data.py     … 各ページの本文・模式図の定義（全45件。register(page) で流し込む）
    ├── sims_data.py      … 体験型シミュレーション25件。SIM[file]=(demo_html, script_js) が図解を差し替え
    ├── advanced_data.py  … 「発展」セクション45件。ADV[file]=HTML（.eq/.fr で数式をHTML組版）
    └── patch_index.py    … index.html のカードに個別ページのリンクを張り直す
```

体験型25件: 01〜10, 12, 13, 23, 24, 26, 27, 31, 33, 35, 36, 38, 40, 42, 43, 45。

## 絶対に守る制約（design lock）

- **完全オフライン・自己完結**：外部CDN・外部画像・外部フォント・外部API・`fetch()` は一切禁止。
  CSS/JS/データは各HTMLに**インライン埋め込み**。`file://` 直開きで全機能が動くこと。
- **1受賞=1ページ**、各ページ**図解は1個**（スライダー2〜4本・ボタン程度の操作）。
- **やさしい本文は必ず残す**（2段構えを崩さない）。発展・シミュレーションを足すのはよいが、
  本文を専門化して置き換えない。
- 体験型シミュレーションは**物理式に基づく実単位の計算**とし、簡略化・規格化した点は
  各図の `.note` に明記する。模式図の回は従来どおり「模式図です」の注記を付ける。
  いずれも**物理的な誤解を招かない**こと。素のJS+Canvas/SVGで軽量に（数値積分は数百点まで）。
- **日本語フォント**はシステム標準を指定（`"Yu Gothic", ... "Meiryo"`）。文字化けさせない。
- 難しい用語は `.term`（点線＋ホバー吹き出し）と、かっこ書きのふりがなで補足する。
- **他プロジェクトのフォルダには触れない**（このフォルダ配下のみ）。

## デザインの要点

- ライトテーマ。ヒーローは分野アクセント色のグラデーション。本文は 背景→発見→意義 の3ブロック。
- 分野=アクセント色：放射線・原子=橙 `#c2410c`／量子力学=紫 `#7c3aed`／素粒子・核=青緑 `#0d9488`／
  宇宙・天文=青 `#2563eb`／物性・応用=緑 `#16a34a`（`build_pages.py` の `THEME`）。
- 個別ページの共通シェルは `build_pages.py` の `TEMPLATE`（プレースホルダ `@@...@@` を置換）。
- 前へ/次へのナビは `NAV`（年代順リスト）から**自動生成**。順番変更は `NAV` の並べ替えだけでよい。

## 生成・再生成のしかた

編集場所: やさしい本文・模式図=`pages_data.py`／シミュレーション=`sims_data.py`／発展=`advanced_data.py`。
編集後に次を実行：

```
python build/build_pages.py     # laureates/ の45ページを生成（SIM/ADV を apply_overrides() で合流）
python build/patch_index.py     # index.html の各カードに個別ページのリンクを張り直す
```

- 共通デザインを直すなら `build_pages.py` の `TEMPLATE`（1か所直せば全45ページに反映）。
  数式は `.eq`（ブロック）と `.fr`（CSS分数）、シミュ操作パネルは `.cpanel`、結果表示は `.rpanel`。
- `pages_data.py` は 01〜45 の全件を定義（手書きだった 01・02・03・06 も取り込み済み。
  SIM に同名キーがあると図解部分はそちらで上書きされる点に注意）。
- 生成後の検証: 各ページの `<script>` を抽出して `node --check`（構文）＋DOMスタブ実行で
  ランタイムエラーを確認できる（2026-07 の改修時に全45件パス済み）。

## 事実・出典の方針（要注意＝outside frontier）

- 事実（受賞年・受賞者・共同受賞者・受賞理由・研究内容）は**ノーベル財団公式サイト（nobelprize.org）**準拠。
  変更・追加時は**人名・年・共同受賞者・citation を必ず要確認**（知識だけに頼らない）。
- 受賞理由は英語 citation の**平易な和訳・要約**（逐語訳ではない）。
- 共同受賞で分野が分かれる年（例：2002 小柴/デービス=ニュートリノ＋ジャコーニ=X線、
  2009 カオ=光ファイバー＋ボイル/スミス=CCD、2018/2020/2021）は本文で分担を切り分ける。
